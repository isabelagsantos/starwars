# Análise Exploratória de Dados — Base Star Wars

## Objetivo

Este projeto tem como objetivo aplicar técnicas iniciais de análise
exploratória, limpeza e visualização de dados utilizando R e RMarkdown.

O estudo foi desenvolvido como exercício prático de ciência de dados,
com foco em organização de bases, interpretação estatística e produção
de relatórios reprodutíveis.

O conjunto de dados utilizado corresponde à base **Star Wars**,
disponível no pacote tidyverse.

------------------------------------------------------------------------

## Lista de variáveis

Essa base de dados contém as seguintes variáveis (e respectivas
traduções):

- name (nome)
- height (altura)
- mass (massa/peso)
- hair_color (cor do cabelo)
- skin_color (cor da pele)
- eye_color (cor dos olhos)
- birth_year (ano de nascimento)
- sex (sexo biológico)
- gender (gênero/identidade)
- homeworld (planeta natal)
- species (espécie)
- films (filmes)
- vehicles (veículos)
- starships (naves estelares)
